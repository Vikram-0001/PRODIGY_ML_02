# 📊 Customer Segmentation using K-Means Clustering

This project implements a machine learning application that segments customers based on their annual income and spending score using the **K-Means Clustering** algorithm. It includes both a Jupyter Notebook for experimentation and a Flask web app for interactive visualization.

---

## 🚀 Features

- Exploratory Data Analysis (EDA) on customer demographics
- K-Means clustering for customer segmentation
- Elbow Method to determine optimal number of clusters
- Web-based interface for visualizing cluster results
- User-friendly interface powered by Flask

## 📊 Dataset Information

**Dataset**: `Mall_Customers.csv`

**Features:**
- `CustomerID`: Unique ID for each customer
- `Gender`: Male or Female
- `Age`: Age of the customer
- `Annual Income (k$)`: Yearly income in thousands
- `Spending Score (1–100)`: Score assigned based on customer behavior
